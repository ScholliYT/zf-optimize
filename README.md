## Docker
### Docker-compose
Start all containers:  
`docker-compose up --build -d`

Delete all containers:  
`docker-compose down --rmi local`

Find the webapp at: https://localhost:PORT_ASSIGNED_BY_DOCKER/. Check PORT_ASSIGNED_BY_DOCKER with `docker ps`.
